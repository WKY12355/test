## 4 推理主机配置

| 组件 | 要求 |
|---|---|
| GPU | NVIDIA RTX 4090（≥24 GB VRAM） |
| 备注 | 需预留 2×8-pin 供电与 3 槽厚度空间 |

---

## 5 注意事项
1. 所有「→桌子前沿距离」均以机械本体最外侧投影点测量；
2. 中间相机光轴建议垂直向下，偏差 ≤±2°；
3. 打印件推荐 PETG 或 ABS，层高 0.2 mm，填充 ≥30 %；
4. 若桌面厚度 ≥25 mm，可直接用 M6 螺纹 inserts；否则需加装过渡板；
5. 推理主机请单独接地，避免相机 USB 接口因电位差掉线。
